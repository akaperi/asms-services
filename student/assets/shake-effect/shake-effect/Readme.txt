Usage: 
------
 
1) Download and unzip the shake-effect.zip file. 
2) Extract the zip file which contains, four files named "index.html, README.txt and style.css" and js folder will be derived. 
3) Now run the index.html file in your browser to view the functionality of the script. 

style.css: 
----------

1) style.css file contains the 'CSS properties' of the script. 
2) Copy and paste the style elements into your style sheet. 
3) CSS properties like 'color, position, height, width, border,font-size' etc., can be modified according to your needs. 
Note: Before placing the style sheet code, make sure to verify the class name and id names to avert the duplication. 

Index.html: 
----------- 

This file contains the entire functionality of ‘Shake Effect Script’. 

Js Folder:
----------

1) This folder contains two files namely jquery.js and script.js.
2) jquery.js file contains the jquery library functions that supports the jquery animations. Do not change the code in this file. If changes are done in this file, then it will affect the functionality of the script.
3)script.js file contains the  shake-effect  animation effect of the script.


How to embed the script in to your webpage? 
--------------------------------------------

1) Open the index.html file. 
2) The design code is available inside the <body> of the Index.html file. Copy the code within the <body></body> tag and paste it inside the <body></body> tag portion of your page.
3) The script and style code is available inside the script.js file and style.css file.Copy the js code, css code and paste it inside  the <script></script> tag and  <style></style> tag portion of your page.
4) Make sure the style.css, jquery.js files and script.js are placed in appropriate folder.
